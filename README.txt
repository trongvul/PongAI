LOS GEHT'S

Python-Version 3.8.2 ist für dieses Projekt erforderlich

Erzeugen einer virtuelle Umgebung
virtualenv venv

Aktivieren der virtuellen Umgebung
source venv/bin/activate

Installation der nötigen Paketen
pip install -r requirements.txt

Starten
python3 main.py 

Deaktivieren der virtuellen Umgebung
deactivate

Für die weiteren Informationen finden Sie die Dokumentation im Verzeichnis docs.
